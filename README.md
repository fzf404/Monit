# Monit

> 用于实时监控自己的 Github 信息
> 
> 使用 vue + electron 开发
> 
> [下载](https://github.com/fzf404/Monit/releases)

<img src="./show.jpeg" alt="首页展示图" style="zoom: 50%;" />

## 介绍

> 每 60s 发送一次请求

1. 左上角第三个按钮的作用是`置顶/取消置顶`。
2. 左侧依次为用户获得的`follower、star、fork`总数
3. 当数字变动时点击可查看变动详情。
4. 右侧为全部仓库及`star`数，点击可跳转到仓库详情，仓库列表可以滚动。
